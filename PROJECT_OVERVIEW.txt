🌐 API ENDPOINTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Public:
    GET    /api/services              List all services
    GET    /api/timeslots             List available time slots
    POST   /api/appointments          Book appointment
  
  Auth:
    POST   /api/auth/login            Admin login
  
  Admin (Protected):
    GET    /api/admin/appointments    List appointments
    PATCH  /api/admin/appointments/:id Update status
    GET    /api/admin/clients/:email  Client history
    CRUD   /api/admin/services        Manage services
    CRUD   /api/admin/timeslots       Manage time slots
